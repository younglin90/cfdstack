## 1. 1D 문제 (Euler)

### 1.1 Sod Shock Tube
- **출처**: Sod, G.A., JCP 27, 1978
- **방정식**: 1D Euler
- **해석해**: 있음 (Riemann solver)
- **도메인**: x ∈ [0, 1], 격막 위치 x = 0.5
- **초기조건**:
  ```
  좌측 (x < 0.5): ρ = 1.0,  u = 0.0,  p = 1.0
  우측 (x ≥ 0.5): ρ = 0.125, u = 0.0, p = 0.1
  ```
- **종료 시간**: t = 0.2
- **γ**: 1.4
- **해석해 구조** (t=0.2):
  ```
  영역 1 (x < 0.263): 원래 좌측 상태
  영역 2 (0.263 < x < 0.486): 팽창파 내부 (자기유사 해)
    ρ = ρ_L · (2/(γ+1) + (γ-1)/((γ+1)·a_L)·(x_0-x)/t)^(2/(γ-1))
    u = 2/(γ+1) · (a_L + (x-x_0)/t)
    p = p_L · (ρ/ρ_L)^γ
  영역 3 (0.486 < x < 0.685): 팽창파 뒤 ~ 접촉면
    ρ ≈ 0.4263, u ≈ 0.9274, p ≈ 0.3031
  영역 4 (0.685 < x < 0.850): 접촉면 ~ 충격파
    ρ ≈ 0.2656, u ≈ 0.9274, p ≈ 0.3031
  영역 5 (x > 0.850): 원래 우측 상태
  ```
- **비교 기준**: ρ, u, p 프로파일. 충격파 위치 x ≈ 0.850, 접촉면 x ≈ 0.685, 팽창파 꼬리 x ≈ 0.263
- **판정**: 충격파 위치 오차 < 1 cell, 접촉면 smearing ≤ 5 cells (2nd order)
- **용도**: flux 함수 기본 검증, Riemann solver 검증

### 1.2 Lax Problem
- **출처**: Lax, P.D., 1954
- **방정식**: 1D Euler
- **해석해**: 있음
- **도메인**: x ∈ [0, 1], 격막 위치 x = 0.5
- **초기조건**:
  ```
  좌측: ρ = 0.445, u = 0.698, p = 3.528
  우측: ρ = 0.5,   u = 0.0,   p = 0.571
  ```
- **종료 시간**: t = 0.16
- **γ**: 1.4
- **용도**: 보다 강한 충격파에서의 flux 함수 검증

### 1.3 Shu-Osher Problem
- **출처**: Shu, C.-W. & Osher, S., JCP 83, 1989
- **방정식**: 1D Euler
- **참조해**: 고해상도 수치해 (N=2000 이상)
- **도메인**: x ∈ [-5, 5]
- **초기조건**:
  ```
  좌측 (x < -4): ρ = 3.857143, u = 2.629369, p = 10.33333
  우측 (x ≥ -4): ρ = 1.0 + 0.2·sin(5x), u = 0.0, p = 1.0
  ```
- **종료 시간**: t = 1.8
- **γ**: 1.4
- **비교 기준**: 밀도 프로파일의 고주파 진동 해상도
- **용도**: 고차 스킴(WENO, compact)의 해상도 검증. 1차 스킴으로는 진동이 사라짐.

### 1.4 Stationary Contact Discontinuity
- **방정식**: 1D Euler
- **해석해**: 초기조건 그대로 (정지 상태)
- **도메인**: x ∈ [0, 1]
- **초기조건**:
  ```
  좌측: ρ = 1.0, u = 0.0, p = 1.0
  우측: ρ = 0.1, u = 0.0, p = 1.0
  ```
- **종료 시간**: 임의
- **비교 기준**: 밀도 점프가 움직이지 않아야 함. 압력/속도 변동 < 1e-10.
- **용도**: flux 함수의 접촉면 보존 능력 검증. AUSM 계열에서 특히 중요.

### 1.5 Slowly Moving Contact
- **방정식**: 1D Euler
- **초기조건**:
  ```
  좌측: ρ = 1.0, u = 0.1, p = 1.0
  우측: ρ = 0.1, u = 0.1, p = 1.0
  ```
- **용도**: 저속 접촉면에서의 과도한 수치 확산 검증

## 2. 2D 문제 (Euler)

### 2.1 Isentropic Vortex Advection
- **출처**: Yee, H.C. et al., JCP 162, 2000
- **방정식**: 2D Euler
- **해석해**: 있음 (초기 조건이 시간에 따라 이류)
- **도메인**: [0, 10] × [0, 10], 주기 경계
- **초기조건**:
  ```
  (u, v) = (1, 1) + (ε/2π) · exp((1-r²)/2) · (-ȳ, x̄)
  T = 1 - (γ-1)ε²/(8γπ²) · exp(1-r²)
  ρ = T^(1/(γ-1)),  p = ρ^γ
  
  x̄ = x - x_c, ȳ = y - y_c, r² = x̄² + ȳ²
  ε = 5.0 (vortex strength), (x_c, y_c) = (5, 5)
  ```
- **비교 기준**: 1주기(t=10) 후 초기조건과의 L2 오차
- **용도**: 공간 스킴의 수렴 차수 정밀 측정. **가장 깔끔한 OOA 테스트.**

### 2.2 Double Mach Reflection
- **출처**: Woodward, P. & Colella, P., JCP 54, 1984
- **방정식**: 2D Euler
- **참조해**: 고해상도 수치해
- **도메인**: [0, 4] × [0, 1]
- **초기조건**: Ma=10 충격파가 30도 경사면에 반사
  ```
  충격파 위치: x = 1/6 + y/√3
  Pre-shock:  ρ = 1.4, u = 0, v = 0, p = 1
  Post-shock: ρ = 8.0, u = 8.25·cos(π/6), v = -8.25·sin(π/6), p = 116.5
  ```
- **비교 기준**: Mach stem 높이, 제트 구조의 해상도, 슬립 라인의 불안정성 포착
- **용도**: 강한 충격파 상호작용에서의 스킴 강건성

### 2.3 Rayleigh-Taylor Instability
- **방정식**: 2D Euler + 중력
- **도메인**: [0, 1/4] × [0, 1]
- **초기조건**:
  ```
  ρ = 2 (y > 0.5), ρ = 1 (y ≤ 0.5)
  p = p_0 - ρ·g·y (정수압 평형)
  v = v_pert · (1 + cos(8πx)) · (1 + cos(5πy)) / 4  (y ≈ 0.5 근처)
  ```
- **비교 기준**: 불안정 성장률, 스파이크/버블 대칭성
- **용도**: 접촉면 해상도 + 소스항 균형 (well-balanced) 검증

## 3. 비압축성 문제

### 3.1 Lid-Driven Cavity (2D)
- **출처**: Ghia, U. et al., JCP 48, 1982
- **방정식**: 비압축 NS
- **도메인**: [0, 1] × [0, 1]
- **경계조건**: 상단 u = 1 (이동벽), 나머지 no-slip
- **Re**: 100, 400, 1000, 3200, 5000, 10000
- **비교 기준**: 중심선 속도 프로파일 (Ghia et al. Table I, II)
  - x=0.5 라인의 u(y) 프로파일
  - y=0.5 라인의 v(x) 프로파일
- **판정**: 주 와류 중심 위치, Ghia 데이터와 1% 이내 일치
- **용도**: 비압축성 솔버의 기본 검증

### 3.2 Taylor-Green Vortex (2D)
- **출처**: Taylor, G.I. & Green, A.E., 1937
- **방정식**: 비압축 NS
- **해석해**: 있음
  ```
  u(x,y,t) = -cos(x)·sin(y)·exp(-2νt)
  v(x,y,t) =  sin(x)·cos(y)·exp(-2νt)
  p(x,y,t) = -(cos(2x) + cos(2y))·exp(-4νt) / 4
  ```
- **도메인**: [0, 2π] × [0, 2π], 주기 경계
- **용도**: 시간 적분 + 공간 이산화 정밀도 동시 검증. **수렴 차수 측정에 이상적.**

### 3.3 Flow Past Circular Cylinder (2D)
- **출처**: 다수 (Williamson, 1996 리뷰)
- **방정식**: 비압축 NS
- **Re**: 20 (정상), 40 (정상, 쌍와류), 100 (비정상, Karman vortex)
- **비교 기준**:
  - Re=20: wake 길이 L/D ≈ 0.94, Cd ≈ 2.05
  - Re=40: wake 길이 L/D ≈ 2.13, Cd ≈ 1.52
  - Re=100: St ≈ 0.164, Cd_mean ≈ 1.33 ± 0.05, Cl_rms ≈ 0.23
- **용도**: 비정상 유동 + 물체 주위 유동의 종합 검증

## 4. 압축성 점성 문제

- **해석해**: 있음 (선형 속도 프로파일 + 포물선 온도 프로파일)

## 5. 난류 문제

### 5.1 Turbulent Flat Plate
- **출처**: Wieghardt, K. & Tillmann, W., 1951 / NASA Langley 데이터
- **방정식**: RANS + 난류 모델
- **Re_x**: 1e6 ~ 1e7
- **비교 기준**: Cf(x), 속도 프로파일 (law of the wall)
- **용도**: 난류 모델 기본 검증

### 5.2 Backward-Facing Step
- **출처**: Driver, D.M. & Seegmiller, H.L., AIAA J. 1985
- **Re_h**: 36,000
- **비교 기준**: 재부착 길이 x_r/h ≈ 6.26, 재부착점 하류 Cf 분포
- **용도**: 분리/재부착 유동에서 난류 모델 성능 검증

### 5.3 Taylor-Green Vortex (3D, DNS)
- **출처**: Brachet, M.E. et al., JFM 130, 1983
- **방정식**: 3D NS (DNS, 난류 모델 없음)
- **Re**: 1600
- **초기조건**:
  ```
  u = sin(x)·cos(y)·cos(z)
  v = -cos(x)·sin(y)·cos(z)
  w = 0
  p = p_0 + (cos(2x) + cos(2y))·(cos(2z) + 2) / 16
  ```
- **도메인**: [0, 2π]³, 주기 경계
- **비교 기준**: 운동 에너지 소산율 -dE_k/dt 시간 이력 (peak 시점 t ≈ 9)
- **용도**: 고차 스킴의 3D 비정상 해상도 + 에너지 보존 검증

## 사용법

각 벤치마크 케이스를 실행할 때:
1. 위 초기조건/경계조건을 **그대로** 사용한다 (임의 변경 금지)
2. 비교 기준에 명시된 물리량을 추출한다
3. `/cfd-vv`의 수렴 차수 측정 또는 벤치마크 비교 프로토콜을 따른다
4. 결과를 기록할 때 케이스 번호를 참조한다 (e.g., "벤치마크 1.1 Sod 결과")
